# FORM-HIJAU
Form Hijau adalah platform pengumpulan ide, laporan, dan aspirasi terkait pelestarian lingkungan. Melalui web ini, pengguna dapat mengirimkan gagasan kreatif, melaporkan masalah lingkungan di sekitar mereka, ataupun memberikan solusi hijau yang dapat diterapkan di sekolah, komunitas, maupun wilayah tempat tinggal.  Dengan tampilan yang sederhana.
